Sobre o Curso

Nossa Metodologia

este curso foi desenvolvido para ensinar HTML, CSS e JavaScript de forma prática e objetiva, permitindo que você comece a criar sites profissionais desde o primeiro dia

Objetibos do Curso

  • Compreender a estrutura básica de HTML e suas tags semânticas
  • Dominar os fundamentos do CSS para criar novos layouts modernos
  • Desenvolver sites resonsivos que funcionam em qualquer dispositivos
  • Criar interações dinâmicas cm JavaScript
  • Implementar as melhores práricas de desenvolvimento web

Público-Alvo

Este curso é ideal para:

  • Estudantes de design e programação
  • Profissionais de marketing e comunicação
  • Empreendedores e autônomos
  • Qualquer pessoa interessada em aprender a criar sites

Estrutura do Curso

Manhã (4 horas)

  • Introdução ao HTML
  • Estrutura básica de páginas
  • Fundamentos do CSS
  • Layouts responsivos

Tarde (4 horas)

  • JavaScript básico
  • Manipulação de DOM
  • Eventos e Interatividades
  • Projetos práticos
